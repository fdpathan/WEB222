Explain the concept of Responsive Design. What does the term mean?

Responsive web design (RWD) is a web development technique that generates dynamic changes to the appearance of a website based on the screen size and orientation of the device used to view it. RWD uses breakpoints to determine how a site's layout will look: one design is used above a breakpoint, and another design is used below that breakpoint. The width of the browser is widely used to estimate breakpoints. Page elements reshuffle in responsive design as the viewport expands or contracts. A three-column desktop layout may be reduced to two columns for a tablet and one column for a smartphone. To rearrange content and design elements, responsive design employs proportion-based grids.
